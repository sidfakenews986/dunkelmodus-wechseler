# Dunkelmodus-Wechsler

## Beschreibung
Dies ist eine einfache Webanwendung, mit der Benutzer schnell zwischen einem Dunkelmodus und einem Lichtmodus für ihre bevorzugten Webseiten umschalten können. Die Anwendung unterstützt auch benutzerdefinierte Farbschemata, um den persönlichen Vorlieben der Benutzer gerecht zu werden.

## Features
- Schneller Wechsel zwischen Dunkelmodus und Lichtmodus
- Unterstützung für benutzerdefinierte Farbschemata
- Speichern von Benutzereinstellungen im lokalen Speicher
- Kompatibel mit allen modernen Browsern

## Installation
Um die Anwendung lokal auszuführen, klonen Sie das Repository und öffnen Sie die `index.html`-Datei in Ihrem Webbrowser.

```bash
git clone https://github.com/IhrBenutzername/dunkelmodus-wechseler.git
cd dunkelmodus-wechseler
open index.html
```

## Verwendung
1. Starten Sie die Anwendung.
2. Wählen Sie Ihre bevorzugte Farbschemata und aktivieren Sie den Dunkelmodus oder Lichtmodus nach Wunsch.
3. Ihre Einstellungen werden im lokalen Speicher gespeichert und beim nächsten Laden der Seite automatisch angewendet.
